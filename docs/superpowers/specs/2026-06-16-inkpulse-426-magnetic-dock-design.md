# InkPulse 4.26" 磁吸侧挂外壳 — 设计文档

- 日期：2026-06-16
- 状态：已确认，开始实现
- 目标：为 4.26" 竖向墨水屏 **YRD0426BBS770F0** 设计一个可 3D 打印的外壳，通过一个 **L 形卡钉支架**磁吸耦合在塑料显示器的**左上角外侧**，竖屏、正对用户、不遮挡画面。本版 **v1 无电池**，纯 Type-C 供电。

## 1. 背景与依据

- **墨水屏**：深圳英瑞达 **YRD0426BBS770F0**（4.26" AM EPD，480×800，1-bit 黑白）。规格书：`hardware/docs/YRD0426BBS770F0HP-V2.0-4.2_Inch-480x800.pdf`。驱动 IC **SSD1677**，4 线/3 线 SPI，24-pin FPC。
  - 外形：**62.37 (W) × 105.33 (H) × 1.0 ±0.10 (厚) mm**。天然竖向。
  - 可视区 (AA)：**55.68 (W) × 92.80 (H) mm**，像素间距 0.116mm。
  - 黑边：水平方向总余量 6.69mm（居中 → 左右各 ≈3.35mm）；垂直方向总余量 12.53mm，**FPC 侧（底部短边）黑边偏宽、AA 上偏**。精确上/下黑边数值规格书机械图未给清晰数字，**列为待测**（见 §8）；CAD 中视窗纵向偏置设为变量。
  - **FPC 排线**：从**底部短边**引出（驱动 IC SSD1677 绑定在底边）。FPC 露出/弯折段长度、宽度、补强分布以实物为准（规格书 REV B 注明"ADD FPC LENGTH"），**待测**。
  - 背面平整，可整面贴泡棉。
- **驱动板（新做小板，外壳为基准、板跟随）**：本设计未指定现成板，由外壳约束反向定义封装包络（见 §5）。控制器 ESP32-S3，屏驱动 SSD1677 4 线 SPI，温湿度可选沿用 HTU21D，USB-C 供电/烧录。
- **显示器（用户实测）**：塑料外壳，左上角 W_top ≈ W_left ≈ **7mm** 黑边；边缘总厚 T_edge ≈ **17mm**；背面靠边缘有**数厘米**平整段（后舌可扣）。
- **约束/选择**（与用户确认）：
  - 安装几何：**方案 C** — 屏体探出显示器左上角**外侧**，几乎不遮挡画面。
  - 屏体与显示器面**共面、正对用户**，不做偏转角度。
  - 支架→显示器固定：**L 形卡钉抱边 + VHB 背胶**。
  - 屏体→支架对接：**A2 纯磁面**（对磁、可摘可上下挪），高摩擦垫抗下滑。
  - **v1 无电池**，纯 Type-C 供电；电池作为后续改版。
  - **Type-C 位于屏体与显示器相邻的侧边（右侧），朝显示器方向出线**，线缆绕至显示器背后藏线。
  - 材料默认 PLA/PETG，层高 0.2mm，3 圈壁。
  - **外壳为机械基准，PCB 跟随**（与现有 7.5" 项目同一哲学）。

## 2. 总体架构

两个 FDM 打印件 + 一组磁铁：

| 件 | 作用 | 打印姿态 |
|---|---|---|
| 屏体（前框 + 后盖） | 前框露视窗压黑边；后盖嵌磁、压合屏与小板；底部 FPC 折回 | 前框正面朝下、后盖平放 |
| L 支架 | 卡钉抱住显示器左上角；伸出对接面嵌对磁 + 高摩擦垫 | 平放 |

装配关系：屏从背面放入前框 → 小板贴屏背、FPC 由底部短边折回接 24P 排座 → 后盖（嵌磁）自攻螺丝压合成屏体组件 → L 支架卡钉抱住显示器左上角、VHB 防滑 → 屏体后盖磁面贴合支架对接面，落在角外侧。屏体与支架间纯磁吸，可随手摘、可沿对接面上下微调。

## 3. 屏体（前框 + 后盖）

- 屏容腔：62.37×105.33，单边间隙 0.4mm。
- **视窗开口**：AA 55.68×92.80 + 0.5mm 余量；水平居中，**纵向上偏**（远离 FPC 侧），偏移量随实测黑边定，CAD 设为变量 `WINDOW_OFFSET_Y`。
- 前框唇边压黑边、不碰 AA，**唇厚 1.0mm**（屏仅 1.0mm 厚，唇做薄避免压伤）。
- 壁厚 2.0mm；屏体外形约 **67 (W) × 110 (H) mm**（派生自屏容腔 + 2×壁）。
- **厚度叠层（v1 无电池，Z 向派生公式）**：前面板/唇 1.0 + 屏 1.0 + FPC 折回间隙 0.5 + PCB 1.6 + 元件 3.5（实测 PCB1.step 最高 ≈3.25 + 余量）+ 元件到后盖余隙 0.3 + 后盖壁 1.5 ≈ **屏体总厚 ~9.4mm**；整机重 ~40g。
- 前框 = **托盘**：实心块从背面挖屏容腔，**四周侧壁（2mm）整条由前框出**（罩住整个侧面）；Type-C 等侧面开口须切在前框侧壁。
- 后盖：与屏体外形齐平，嵌磁腔（见 §4）+ 插入凸台 plug（边框压 PCB 背面四周、中央镂空让过元件）；4× M2 自攻压向前框，同时夹紧 PCB（无螺丝夹持，见 §5）。
- **FPC 过线**：底部短边中央开槽，FPC 折回（保证最小弯折半径，PI 补强段直出、软段折弯），槽宽对齐 FPC 实测宽 + 余量。
- **Type-C 开口**：切在**前框右侧壁**（朝显示器方向）；竖向居于上下两排磁的空带、屏顶下方约 27mm（参数 `TYPEC_CENTER_Y=-27`，避开磁腔）；开口中心 Z 由板位推出（见 §5）；开口 ~10×4mm（含容差）。已用"装配后屏体合并实体 ∩ 穿透探针 = 0"校验为真通孔。

## 4. L 支架 + 磁吸对接

- **卡钉抱边（U 形口）**：抱住显示器顶边与左边；U 口深 ≈ **17mm**（= T_edge，留 +0.5 弹性余量）；**前舌 ≤5mm**（只压 7mm 黑边、绝不盖画面）；**后舌伸入背面 ≈8mm** 扣住承重。抱边内侧贴 **VHB 双面胶**防滑防脱。
- **支柱（直支柱）**：从顶边卡钉根部沿屏体顶边**笔直向 −X 外延**，全程藏在屏体平面之后（Z<显示器前表面）、Y 在屏顶高度一带（约 Y∈[−12,0]），把对接面送到画面之外、使屏体落位。截面 `STRUT_W×STRUT_H`（默认 12×8mm，保刚度），末端整截面埋入对接板成加强背肋（宽熔接、单一连通实体）。支柱在屏体左/后，Type-C 在右侧边，二者天然错开不干涉。
- **磁吸对接面（A2 纯磁面）**：支架对接面与屏体后盖**各嵌一组对磁（N52）**，吸引极性排布 → 自动吸正、有"咔哒"定位感、可沿面上下微调。
  - 默认 **4 × Φ8×3mm N52**（上下各一对），按 ~40g 屏体 + 杠杆力，经后盖 1.5mm 壁 + 气隙后仍留 ≳3× 抗剪余量；定稿前按重量复算颗数/磁径。
  - 对接面贴 **高摩擦 TPU/硅胶垫**，把"久放下滑"风险压到最低（A2 承重靠磁摩擦，故必加）。
  - 磁铁嵌入腔用压配或胶固定，极性方向在装配文档标注，防装反相斥。

## 5. PCBA 装配约定 v1（外壳基准，PCB 跟随）

坐标原点 = 板几何中心；X = 屏宽方向（右为正，右=朝显示器）；Y = 屏高方向（上为正）；Z = 厚度（前=朝用户）。单位 mm。

**装配方式（无螺丝夹持）**：PCB 在屏后、**元件/连接器面朝后盖**、屏面侧平整。前框内 **4 根支撑支柱**把板顶到 Z 平面（板前面 z=2.5），其中 **2 根对角带定位销**（Φ1.8）插板孔做 XY 精定位、其余靠腔壁兜底（板外形 = 腔内 −0.3）。后盖 **plug 凸台边框**从背面压 PCB 四周边沿（中央镂空让过元件，留 0.3 余隙），后盖 4× M2 自攻拧紧即夹住板（约 0.1 预压）。

| 项目 | 规格 | 位置（板中心原点） |
|---|---|---|
| 板外形 | 名义 **58 (W) × 90 (H)**（≤包络 60×95），板厚 1.6 | 藏于屏正后方、居中 |
| 定位/夹持 | 2 对角定位销插孔 + 4 支柱托 + 后盖 plug 压边沿 | 角部内缩、避开连接器 |
| 最高元件 | ≤ **3.5mm**（实测 PCB1.step ≈3.25 + 余量），**朝后盖侧** | — |
| 24-pin 显示排座 | 0.5mm FPC 插座，**元件面（朝后盖）** | **底边中点 (0, −H/2)**，对位屏 FPC 折回 |
| Type-C 母座 | 供电/烧录，**元件面**，体高出板 ~3.2 | **右侧边 (X+ 边)**，口朝 +X；开口中心 Z = PCB背面(z≈4.1)+体高/2 ≈ **5.7**；Y= 屏顶下 27mm（磁排空带） |
| 主控/屏驱 | ESP32-S3 + SSD1677 4 线 SPI（SCL/SDA/CS/DC/RES/BUSY） | — |
| 传感器（可选） | HTU21D 温湿度 I2C | — |

> **连接器朝向**：24P FPC 排座与 USB-C 均置于元件面（朝后盖），屏面侧平整使屏可贴近。屏 FPC 从底部短边折回、绕到背面插排座。
> **USB-C 触及**：板右缘到前框侧壁内面留有 ~1.6mm 间隙；真机布板时把板右缘对齐腔壁（或用略外伸的边缘式 USB-C），确保口面对准开口。
> **v1 不含**电池、充电管理、电池保护 IC。Type-C 仅供电 + 烧录。电池改版（后续）：后盖加厚出电池仓变体 + 板上加 TP4056/DW01，接口约定向后兼容。

## 6. 打印工艺与公差

- 前框正面朝下、后盖与支架平放，目标免支撑或极少支撑。
- 配合间隙 0.4mm；自攻螺纹底孔按 M2 自攻留 0.2mm。
- 材料 PLA/PETG，层高 0.2mm，壁 3 圈。
- 输出：**STEP**（装配/干涉检查）+ **STL**（切片打印）。

## 7. 实现方式（工具链）

- **build123d（Python）** 参数化建模，所有尺寸为脚本顶部变量（沿用现有 `cad/` 做法）。
- 环境：用户提供的 conda 环境（含 build123d）；**缺依赖时不得 pip 安装，须向用户索取环境**。
- 校验：装配干涉、最小弯折半径、磁吸抗剪余量、视窗对 AA 偏置；屏体落位不遮挡画面、不撞显示器。
- 重活（建模、校验）派 subagent 执行，结论回传，避免污染主上下文。

## 8. 未决 / 风险

- **AA 纵向黑边偏置未知**：规格书机械图未给清晰上/下黑边数值 → 视窗纵向偏移待实测；CAD 设为变量，不阻塞腔体设计。
- **FPC 尺寸/补强/弯折段未知**：规格书 REV B 仅注"ADD FPC LENGTH" → 露出长度、宽度、最小弯折半径待实测；影响折回槽深与板上排座间距。
- **A2 纯磁面下滑风险**：靠对磁 + 高摩擦垫缓解；装配后须做静置下滑测试，必要时加磁或加深摩擦垫纹理。
- **磁吸抗剪余量**：屏体探角外有杠杆力，需按实测重量复算磁力，留 ≥3×。
- **支柱下沿略探出显示器背面**：为保刚度 `STRUT_H=8`，支柱下沿 Z≈−18 比显示器背面 −17 探出约 1mm；在显示器左缘外（X<0）不干涉，仅侧视可见。如需完全藏入可减小 `STRUT_H`/`MATE_GAP`（降刚度）。
- **臂↔对接板腹板/熔接随参数联动**：大改 `MATE_*`、磁位 `MAG_BAND_CENTER_Y` 或支柱几何后，需复核支柱仍整截面咬住对接板（自检会即时报警）。
