# InkPulse 4.26" 磁吸侧挂外壳 — 设计文档

- 日期：2026-06-16
- 状态：已确认，开始实现
- 目标：为 4.26" 竖向墨水屏 **YRD0426BBS770F0** 设计一个可 3D 打印的外壳，通过一个 **L 形卡钉支架**磁吸耦合在塑料显示器的**左上角外侧**，竖屏、正对用户、不遮挡画面。本版 **v1 无电池**，纯 Type-C 供电。

## 1. 背景与依据

- **墨水屏**：深圳英瑞达 **YRD0426BBS770F0**（4.26" AM EPD，480×800，1-bit 黑白）。规格书：`hardware/docs/YRD0426BBS770F0HP-V2.0-4.2_Inch-480x800.pdf`。驱动 IC **SSD1677**，4 线/3 线 SPI，24-pin FPC。
  - 外形：**62.37 (W) × 105.33 (H) × 1.0 ±0.10 (厚) mm**。天然竖向。
  - 可视区 (AA)：**55.68 (W) × 92.80 (H) mm**，像素间距 0.116mm。
  - 黑边：水平方向总余量 6.69mm（居中 → 左右各 ≈3.35mm）；垂直方向总余量 12.53mm，**FPC 侧（底部短边）黑边偏宽、AA 上偏**。精确上/下黑边数值规格书机械图未给清晰数字，**列为待测**（见 §8）；CAD 中视窗纵向偏置设为变量。
  - **FPC 排线**：从**底部短边**引出（驱动 IC SSD1677 绑定在底边）。FPC 露出/弯折段长度、宽度、补强分布以实物为准（规格书 REV B 注明"ADD FPC LENGTH"），**待测**。
  - 背面平整，可整面贴泡棉。
- **驱动板（新做小板，外壳为基准、板跟随）**：本设计未指定现成板，由外壳约束反向定义封装包络（见 §5）。控制器 ESP32-S3，屏驱动 SSD1677 4 线 SPI，温湿度可选沿用 HTU21D，USB-C 供电/烧录。
- **显示器（用户实测）**：塑料外壳，左上角 W_top ≈ W_left ≈ **7mm** 黑边；边缘总厚 T_edge ≈ **17mm**；背面靠边缘有**数厘米**平整段（后舌可扣）。
- **约束/选择**（与用户确认）：
  - 安装几何：**方案 C** — 屏体探出显示器左上角**外侧**，几乎不遮挡画面。
  - 屏体与显示器面**共面、正对用户**，不做偏转角度。
  - 支架→显示器固定：**L 形卡钉抱边 + VHB 背胶**。
  - 屏体→支架对接：**POGO 磁吸连接器**（自带 N/S 磁吸合 + 4 针供电/数据），屏体落位即得电。
  - **v1 无电池**，供电经支架进入：**Type-C 在支架底边** → 支架内走线/小板 → POGO → 屏体。屏体本身无任何线缆口。
  - 电池作为后续改版。
  - 材料默认 PLA/PETG，层高 0.2mm，3 圈壁。
  - **外壳为机械基准，PCB 跟随**（与现有 7.5" 项目同一哲学）。

## 2. 总体架构

两个 FDM 打印件 + 8×Φ8 磁铁 + POGO 磁吸连接器：

**磁/电分工**：**8×Φ8 磁铁（后盖 4 + 支架 4）负责吸合/承重**；**POGO 连接器负责供电/数据 + 对位**（其自带 N/S 小磁仅辅助）。二者在对接带上不冲突：POGO 居中央、4 磁在四角。

| 件 | 作用 | 打印姿态 |
|---|---|---|
| 屏体（前框 + 后盖） | 前框露视窗压黑边；后盖对接区 = 4×Φ8 磁腔 + **POGO 配对腔**（成对那半）；压合屏与小板 | 前框正面朝下、后盖平放 |
| L 支架 | 卡钉抱住显示器左上角；对接面嵌 4×Φ8 磁 + POGO 连接器；底边 Type-C 壳块 + 内部走线 | 平放 |

装配关系：屏从背面放入前框 → 小板贴屏背、FPC 由底部短边折回接 24P 排座 → 后盖压合成屏体组件（板背 4 pogo pad + 2 钢片 + 后盖 4 磁露在对接区）→ L 支架卡钉抱住显示器左上角、VHB 防滑 → Type-C 插支架底边、支架内走线接 POGO → 屏体后盖贴合支架对接面，**8 磁吸合承重 + POGO 4 针压上 pad 供电**：落位即耦合 + 得电。可随手摘（断电）、再贴（通电）。

## 3. 屏体（前框 + 后盖）

- 屏容腔：62.37×105.33，单边间隙 0.4mm。
- **视窗开口**：AA 55.68×92.80 + 0.5mm 余量；水平居中，**纵向上偏**（远离 FPC 侧），偏移量随实测黑边定，CAD 设为变量 `WINDOW_OFFSET_Y`。
- 前框唇边压黑边、不碰 AA，**唇厚 1.0mm**（屏仅 1.0mm 厚，唇做薄避免压伤）。
- 壁厚 2.0mm；屏体外形约 **76 (W) × 119 (H) mm**。注：因屏几乎占满屏腔、侧壁仅 2mm 放不下穿腔 M2 螺柱，**边框加宽**使 4 角 M2 螺柱落在墨水屏 footprint **之外**（boss 中心由 `屏半尺寸 + BOSS_SCREEN_CLEAR + 柱半径` 派生，boss∩屏=0）。屏仍居中屏腔、完整包住；墨水屏顶仍 ≈ 显示器顶（加宽的顶边框略高于显示器顶）。
- **厚度叠层（v1 无电池，Z 向派生公式）**：前面板/唇 1.0 + 屏 1.0 + FPC 折回间隙 0.5 + PCB 1.6 + 元件 3.5（实测 PCB1.step 最高 ≈3.25 + 余量）+ 元件到后盖余隙 0.3 + 后盖壁 1.5 ≈ **屏体总厚 ~9.4mm**；整机重 ~40g。
- 前框 = **托盘**：实心块从背面挖屏容腔，**四周侧壁（2mm）整条由前框出**（罩住整个侧面）；Type-C 等侧面开口须切在前框侧壁。
- 后盖：与屏体外形齐平 + 插入凸台 plug。**plug 按 PCB(60×95) 派生**（非屏腔）：rim 落在 [元件包络 .. PCB 边] 的非元件区、**四边都压 PCB 背面边沿**（中央镂空让过元件，`∩元件=0`）；预压 0（靠 4× M2 螺丝夹紧，不靠塑料过盈，`∩PCB=0`）。
- **后盖对接区**：**4×Φ8×3 磁腔**（四角，与支架磁镜像、棋盘极性防呆，承重主力；盖板仅 1.5mm 故内侧长 Φ11.2 磁柱 boss 补料再掏腔、外侧 0.8 薄壁、内侧开口可装入）+ **POGO 配对腔**（POGO 成对使用：支架嵌一半、后盖嵌配对那半；本体凹腔从内侧敞开装入、对接面留 0.8 薄壁 + 4 触点窗 Φ1.6 + 2 安装孔，与支架 POGO 镜像偏差 0）。
- **屏体无 Type-C 开口**（已删除）——供电改由 POGO 经背面 pad 进入。
- **FPC 过线**：屏 FPC 从底部短边折回到背面接板。折回让位槽**暂移除**（`FPC_SLOT_ENABLE=False`）——待 FPC 实测后再收敛形状（§8）。`FPC_FOLD_GAP=0.5` 的 Z 向折回间隙保留。

## 4. L 支架 + POGO 磁吸对接 + 底边 Type-C

- **卡钉抱边（U 形口）**：抱住显示器顶边与左边；U 口深 ≈ **17mm**（= T_edge，留间隙）；**前舌 ≤5mm**（只压 7mm 黑边、绝不盖画面）；**后舌伸入背面 ≈8mm**；抱边内侧贴 **VHB 双面胶**防滑防脱（四面留 0.3 间隙，correct-by-construction）。
- **支柱（直支柱）**：从顶边卡钉根部沿屏体顶边**笔直向 −X 外延**，全程藏在屏体平面之后（Z<显示器前表面）、Y 在屏顶高度一带。截面 `STRUT_W×STRUT_H`（12×8mm 保刚度），末端整截面埋入对接板成加强背肋（宽熔接、单一连通实体）。
- **对接面磁 + POGO（共存）**：对接面 **4×Φ8×3 磁**（四角，与后盖镜像、棋盘极性，**承重主力**）+ 中央嵌 **POGO 磁吸连接器**（实测 20.44×4.0×本体2.0；针凸出 1.5 朝屏；4 针 2.54/7.62；N/S 磁相距 13.54；2× Φ1.5 安装孔相距 17.54），**POGO 负责 4 针供电/数据**（VBUS/GND/D+/D−）+ 对位。POGO 居中、4 磁居四角，互不重叠。
- **底边 Type-C 壳块（供电入口）**：支架对接板底部长出**专用实体壳 boss**（19.9×13.5×Y16.8，四壁 ≥1.6，藏屏后、向 −Z/−Y 凸）完整包裹防水带飞线 Type-C 母座（开孔 13.60×4.30 R1.3 朝下；法兰 16.70×10.30 沉台；本体 14mm 深）。注：对接板仅 5mm 厚，**不可直接切薄板**（法兰 10.3mm 比板厚大一倍），必须用壳块包裹。支架内 **走线腔**（7×4）从 Type-C 体腔通到 POGO，容飞线 + 小 dock PCB（CC1/CC2 下拉电阻）。
- **承重/磁力（重点风险）**：POGO 自带 N/S 磁吸力通常远小于原 4×Φ8 N52，而屏体 ~40g + 探角外杠杆力对抗剥离/抗剪要求高。**装配后必做静置下滑 + 摘挂手感测试**；不足则开 `AUX_MAG_ENABLE=True`（POGO 上下各加一对 Φ6 补强磁，屏体已预留镜像腔），并在对接面双侧贴高摩擦垫。

## 5. PCBA 装配约定 v1（外壳基准，PCB 跟随）

坐标原点 = 板几何中心；X = 屏宽方向（右为正，右=朝显示器）；Y = 屏高方向（上为正）；Z = 厚度（前=朝用户）。单位 mm。

**装配方式（无螺丝夹持）**：PCB 在屏后、**元件/连接器面朝后盖**、屏面侧平整。**后盖内 4 根支撑支柱**把板顶到 Z 平面（PCB 背面 z=4.1；支柱在后盖、不在前框——前框前腔被墨水屏占着，放支柱会戳屏），其中 **2 根对角带定位销**（Φ1.8）插板孔做 XY 精定位、其余靠腔壁兜底（板外形 = 腔内 −0.3）。后盖 **plug 凸台边框**从背面压 PCB 四周边沿（中央镂空让过元件），后盖 4× M2 自攻拧紧即夹住板（约 0.1 预压）。

| 项目 | 规格 | 位置（板中心原点） |
|---|---|---|
| 板外形 | 名义 **58 (W) × 90 (H)**（≤包络 60×95），板厚 1.6 | 藏于屏正后方、居中 |
| 定位/夹持 | 2 对角定位销插孔 + 4 支柱托 + 后盖 plug 压边沿 | 角部内缩、避开连接器 |
| 最高元件 | ≤ **3.5mm**（实测 PCB1.step ≈3.25 + 余量），**朝后盖侧** | — |
| 24-pin 显示排座 | 0.5mm FPC 插座，**元件面（朝后盖）** | **底边中点 (0, −H/2)**，对位屏 FPC 折回 |
| **POGO 接触 pad ×4** | 板背 4 个镀金 pad（受电/数据），对位 pogo 针 | 中上部、2.54 间距/7.62 跨度，露于后盖 4 窗 |
| ~~Type-C 母座~~ | **已从屏体板移除**（改在支架，见 §4） | — |
| 主控/屏驱 | ESP32-S3 + SSD1677 4 线 SPI（SCL/SDA/CS/DC/RES/BUSY） | — |
| 传感器（可选） | HTU21D 温湿度 I2C | — |

> **屏体主板**：去 Type-C 母座；板背增 4 个 POGO 接触 pad（VBUS/GND/D+/D−）；其余（ESP32-S3+SSD1677+FPC）不变。屏体无任何线缆口，落位经 POGO 得电。
> **支架内 dock 小板**：防水 Type-C 飞线母座（6 线 V/D−/D+/G/CC1/CC2）→ CC1/CC2 各 5.1kΩ 下拉（呈 sink）→ VBUS/GND/D+/D− 接 POGO 4 针。
> **v1 不含**电池、充电管理、电池保护 IC。电池改版（后续）：后盖加厚出电池仓变体；POGO 接口向后兼容。

## 6. 打印工艺与公差

- 前框正面朝下、后盖与支架平放，目标免支撑或极少支撑。
- 配合间隙 0.4mm；自攻螺纹底孔按 M2 自攻留 0.2mm。
- 材料 PLA/PETG，层高 0.2mm，壁 3 圈。
- 输出：**STEP**（装配/干涉检查）+ **STL**（切片打印）。

## 7. 实现方式（工具链）

- **build123d（Python）** 参数化建模，所有尺寸为脚本顶部变量（沿用现有 `cad/` 做法）。
- 环境：用户提供的 conda 环境（含 build123d）；**缺依赖时不得 pip 安装，须向用户索取环境**。
- 校验：装配干涉、最小弯折半径、磁吸抗剪余量、视窗对 AA 偏置；屏体落位不遮挡画面、不撞显示器。
- 重活（建模、校验）派 subagent 执行，结论回传，避免污染主上下文。

## 8. 未决 / 风险

- **AA 纵向黑边偏置未知**：规格书机械图未给清晰上/下黑边数值 → 视窗纵向偏移待实测；CAD 设为变量，不阻塞腔体设计。
- **FPC 尺寸/补强/弯折段未知**：规格书 REV B 仅注"ADD FPC LENGTH" → 露出长度、宽度、最小弯折半径待实测；影响折回槽深与板上排座间距。
- **承重（已用 8×Φ8 解决）**：POGO 自带 N/S 小磁不足以扛 ~40g + 杠杆力，故**保留 8×Φ8 磁承重**（POGO 仅供电/对位）。装配后仍建议做静置下滑测试；若仍不足可再开 `AUX_MAG_ENABLE` 或加大磁径/加高摩擦垫。
- **磁柱 boss 占板四角**：后盖为容 3mm 磁腔在内侧长 Φ11.2 磁柱（盖板仅 1.5mm）→ **PCB 四角须避开这 4 个磁柱**（铜面布局人工避让；元件包络自检已确保不撞）。
- **POGO 针—pad 接触压力**：依赖针凸出 1.5 与磁吸气隙 `MATE_GAP`/`CONTACT_AIR_GAP` 的压缩行程；真机需确认 pad 镀金、接触电阻、4 针对位精度（POGO↔pad 镜像偏差当前 0）。
- **支架内 dock 小板/CC 电阻**：仅约定，未建模；Type-C 飞线走向与小板布局待实物确认。
- **支柱下沿略探出显示器背面**：为保刚度 `STRUT_H=8`，支柱下沿 Z≈−18 比显示器背面 −17 探出约 1mm；在显示器左缘外（X<0）不干涉，仅侧视可见。如需完全藏入可减小 `STRUT_H`/`MATE_GAP`（降刚度）。
- **臂↔对接板腹板/熔接随参数联动**：大改 `MATE_*`、磁位 `MAG_BAND_CENTER_Y` 或支柱几何后，需复核支柱仍整截面咬住对接板（自检会即时报警）。
