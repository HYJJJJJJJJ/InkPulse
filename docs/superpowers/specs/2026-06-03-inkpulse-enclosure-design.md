# InkPulse 墨水屏桌面外壳 — 设计文档

- 日期：2026-06-03
- 状态：已确认，待实现
- 目标：为 InkPulse 项目（7.5" 三色墨水屏 + 50×50 驱动板）设计一个可 3D 打印的楔形桌面外壳。

## 1. 背景与依据

- **墨水屏**：深圳英瑞达 **E075A42**（7.5" 三色 AMEPD，800×480）。规格书：`hardware/docs/英瑞达E075A42 （BWR）(1).pdf`。
  - 外形：**170.20 (W) × 111.20 (H) × 1.25 (厚，含膜) mm**，重 44±0.5g。
  - 可视区 (AA)：**163.20 × 97.92 mm**。
  - 黑边（AA 到外形，**非对称**）：上 3.49 / 下 9.79（FPC 侧）/ 左右各 3.50 mm（长边方向居中，短边方向 AA 上偏）。
  - **FPC 排线**：从 **170.20mm 长边居中**引出（判据：偏宽黑边 9.79mm 落在 111.20mm 方向轴上，绑定边垂直于该轴 → 即长边）。FPC 宽 **79.90 ±0.2mm**，露出/弯折段 **11.50–12.50mm**，带 PI 补强（总厚 0.30mm，补强段不可弯，弯折仅发生在 0.10mm 软段）。
  - 背面平整、无金属背板或突起，可整面贴泡棉。
  - 接口 24-pin（脚距规格书未标注数值，需实测/向供应商确认，常见 0.5mm 但不可据此下料）。
- **驱动板**：**70.10 × 50.04mm**（实测自 `hardware/PCB1.step`，由 build123d 导入测量）。Type-C 与显示 FPC 排座分别在两条 70mm 长边中点。板载 16P Type-C 母座用于供电/烧录。
  - 早期为 50×50，后改版为 70×50；外壳已据 STEP 实测更新（见 §5）。
- **约束/选择**（与用户确认）：
  - 屏横放（170.2 宽 × 111.2 高），FPC 在底部长边。
  - 外形为 "∠" 楔形，底座位于屏框**正下方**。
  - 屏面与水平桌面**仰角 60°**。
  - 三件式打印 + 螺丝装配。
  - PCB 四角 M3 螺柱固定。
  - Type-C 开口**朝后**（背向用户）。
  - 打印机幅面 ≥ 220×220（屏框可整片打印）。
  - 材料默认 PLA/PETG，层高 0.2mm，3 圈壁。
  - **外壳为机械基准，PCB 跟随**（见 §5 接口约定）。

## 2. 总体架构

楔形桌面支架，三个 FDM 打印件 + 螺丝装配：

| 件 | 作用 | 打印姿态 |
|---|---|---|
| 前框 bezel | 正面开视窗露可视区，唇边压玻璃黑边 | 正面朝下 → 前表面最平整 |
| 后盖 back cover | 压紧玻璃、保护背面、开 FPC 过线槽 | 平放 |
| 底座 base（楔形） | 装 50×50 PCB，与前框 60° 拼接，后壁开 Type-C 口 | 平放 |

装配关系：玻璃从背面放入前框 → 垫 0.5mm 泡棉 → 后盖螺丝压紧（屏框组件）→ 屏框组件在 60° 斜接缝与底座螺丝拼接 → PCB 用 M3 锁在底座螺柱上 → FPC 从屏框底经后盖线槽折入底座接显示排座。

## 3. 屏框（前框 + 后盖）

- 玻璃容腔：170.20×111.20，单边间隙 0.4mm（实装容差）。
- **视窗开口**：163.20 × 97.92 + 0.5mm 余量；**向上偏置**——上边框 ~3.0 / 下边框（FPC 侧）~9.3 / 左右 ~3.0mm。视窗中心相对玻璃中心**上移 ≈ 3.15mm**（= (9.79−3.49)/2）。
- 前框唇边压黑边（不碰 AA），唇厚 1.2mm。
- 壁厚 2.5mm；屏框外形约 **176 × 117mm**。
- **背腔减薄至 5mm**（仅容玻璃 1.25 + 泡棉 0.5 + 薄后盖凸台）→ 前框总深 **7.2mm**（原 19.2），后盖总厚 **5.25mm**（原 17.25）。
- 后盖：与屏框外形齐平，4× M2.5 自攻螺丝压向前框；玻璃背面垫 0.5mm 泡棉。
- 后盖 FPC 过线槽：**宽 84 × 深 13mm**，位于底部长边中央（对齐 FPC 79.9mm 宽 + 余量）。PI 补强段直出、软段折弯下行。

## 4. 底座（楔形 + PCB 仓 + 宽大底脚板）

- **中央 PCB 核心盒**：内腔容 50×50 PCB，四周间隙 0.5mm；4× M3 打印螺柱（boss），柱心 (±21, ±21)，柱高 4mm；内腔净高 15mm。
- **后壁 Type-C 开口**：10 × 4mm（含 0.5 容差 + 倒角），中心离内底 ≈ **7.3mm**；Type-C 朝 +Y（背向用户）。底脚板低矮（4mm）不遮挡插拔。
- **宽大扁平底脚板（稳定性）**：**130 (W) × 90 (D) × 4mm**，横跨支撑 176mm 宽屏、抗后倾。屏重心水平投影（Y≈−16）落在脚板内 → 不倾倒。
- **两片 60° 斜墙（gusset）**：外移到脚板宽边 **±63**，横撑屏宽；起于脚板顶。**斜面朝 −Y+Z（朝观察者-上），使屏朝 −Y 观察者、向后倾**（朝向经校验：法线 ny=−0.87, nz=+0.50, arccos=60°）。2× M3 接缝螺丝横穿斜墙。
- 顶部线槽对位显示排座，承接屏框下来的 FPC。

## 5. PCB 机械接口约定 v1（外壳基准，PCB 跟随）

坐标原点 = 板几何中心；X 右为正；Y 朝 Type-C/后为正、朝屏/前为负。单位 mm。

**实测自 `hardware/PCB1.step`（板几何中心为原点；X=长边方向，Y=Type-C↔FPC 方向）**：

| 项目 | 规格 | 位置（板中心原点） |
|---|---|---|
| 板外形 | **70.10 × 50.04**，板厚 1.6（STEP 简化为 0.41，取实际 1.6） | 中心对齐外壳 |
| 安装孔 ×4 | **Φ3.2 (M3)** | **(±30, ±20)** |
| Type-C 母座 16P | body 11.3×7.6，口中心离板底 ≈1.65mm | **Y− 长边中点 (0, −22)** |
| 24-pin 显示排座 | 0.5mm 间距 FPC 插座，~12mm 宽 | **Y+ 长边中点 (0, +22)** |
| 最高元件 | **≈3.7mm**（ESP32 模块, 17.6×15.8 @ (−8.7,2.4)） | — |

> **装配朝向**：Type-C 边朝外壳后（+Y），FPC 排座边朝屏（前/−Y）。装入时 PCB 相对外壳绕 Z 翻 180°。
> 外壳侧已据此更新：PCB 仓 71.1×51、螺柱 (±30,±20)、Type-C 开口在 +Y 壁中心高 7.65mm。
> 已用 `cad/fitcheck.py` 将真实 PCB 放入底座校验：螺柱对孔、板落仓内、Type-C 对位开口，全部通过。

## 6. 打印工艺与公差

- 前框正面朝下、后盖与底座平放，目标**免支撑或极少支撑**。
- 配合间隙 0.4mm；自攻螺纹底孔按 M2.5/M3 自攻留 0.2mm。
- 材料 PLA/PETG，层高 0.2mm，壁 3 圈。
- 输出：STEP（导回 Altium 做装配/干涉检查）+ STL（切片打印）。

## 7. 实现方式（工具链）

- **build123d（Python）** 参数化建模，所有尺寸为脚本顶部变量。
- 环境：用户提供的 conda 环境（待建：`conda create -n cad python=3.11 && pip install build123d`）。
- 重活（写脚本、生成模型、干涉/可打印性验证）派 subagent 执行，结论回传，避免污染主上下文。

## 8. 未决/风险

- **24-pin 排座脚距未知**：规格书未标，建议实测；不影响外壳腔体设计（排座位置由 §5 约定），仅影响 PCB 选型。
- **Type-C 母座精确机械尺寸**：按 16P 通用值取默认，待实物/封装确认后微调开口。
- **重心稳定性**：170×111 屏 60° 后倾于 50mm 深底座，需在 CAD 后用质心/倾覆校核；必要时底座向后延伸或加配重槽。
- FPC 软段最小弯折半径需在装配验证时确认不超限。
