一句话:

  • POGO 弹针只凸出 1.5mm,针尖止在 −9.60,刚探进后盖外壁 0.2mm。
  • 受电 pad 却建在 PCB 背面 −4.10,在针尖上方 5.5mm 处。
  • 中间是:空气(2.4) + 空配对腔(2.5) + 半截塑料壁(0.6),全程没有导体。
  • 所以屏体落位时,针压根碰不到 pad → 通不了电。这是"该接触却没接触"的空隙,体积干涉矩阵看不出来(空隙不产生重叠体积)。

注:设计文档本身就有两套互相矛盾的说法——§5 说"针直接接板背 pad",§3/§4 说"后盖嵌配对 POGO 半身来桥接"。但配对半身没建模、配对腔也只有 2.5mm 深,够不到 −4.10 的 pad。两种说法当前几何都不成立。
