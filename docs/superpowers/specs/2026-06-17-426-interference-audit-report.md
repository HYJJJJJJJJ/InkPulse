## 2. 真问题逐条（根因 + 可制造性 + 修复建议）

### 🔴 A — POGO 针尖够不到受电 pad，落位无法得电（功能性致命）

**证据（world Z 锚点，`audit_probe_426.py` §4 + 末段）：**

```
墨水屏前表面 Z0           =  0.00
屏体后盖外表面(贴支架)     = -9.80
对接板顶 mate_z_top        = -10.30
POGO 针基面(凹腔顶)        = -11.10
POGO 针尖 (凸出1.5)        =  -9.60   ← 针能到的最高点
后盖配对腔底(朝PCB一侧)    =  -6.50
PCB 受电 pad              =  -4.10   ← 要接触的目标
针尖→pad Z 间隙            =   5.50 mm   ✗
配对腔底→pad 间隙          =   2.40 mm   ✗
```

**根因（坐标系/叠层冲突，非单行 bug）：**
- 支架侧 POGO 针只凸出 `POGO_PIN_PROTRUDE=1.5`（`enclosure_426.py:99`），针尖止于世界 Z=−9.60，仅探入后盖外表面（−9.80）下方 0.2mm。
- 受电 pad 建在 **PCB 背面**（`make_pcba()` 中 `PCB_BACK_Z=4.1` → 翻转后世界 Z=−4.10，`enclosure_426.py:1088-1092`）。
- 二者之间隔着整个后盖板(1.5)+plug 深(4.2)≈5.7mm 的厚度，POGO 针根本够不到。
- **设计文档自相矛盾**：§5 PCBA 表写"板背 4 pad 露于后盖 4 窗、对位 pogo 针"（暗示针直接接 pad）；§3/§4 又写"配对 POGO（成对，后盖嵌配对那半）"（暗示靠第二个 POGO 桥接）。CAD 同时建了 pad（在 PCB 背）+ 触点窗 + 配对腔，但：
  1. 若按"针直接接 pad"：pad 在 −4.10、针尖在 −9.60，差 5.50mm，**几何上不可能**。
  2. 若按"配对 POGO 桥接"：配对 POGO 半身**未建模为实体**（§3⚠️ 明列），且后盖配对腔只有 2.5mm 深（腔底 −6.50），离 pad（−4.10）仍差 **2.40mm**——桥接件长度没人核算过，凹腔深度也不够把触点送到 PCB pad。

**可制造性结论：** 当前几何下，屏体"落位即得电"无法成立。无论按哪种解读，受电侧接触链都断了。这是**整机功能成败的头号风险**，且前几轮 review 用"导出 STEP pairwise / 只看体积重叠"都看不到（空隙不产生体积）。

**修复建议（择一，需用户拍板架构）：**
- **方案 1（推荐，贴合 §3/§4"配对 POGO"架构）**：在后盖配对腔显式建模配对 POGO 半身实体，确认其"对接触点面（朝支架）→ 焊脚（朝 PCB pad）"的实体跨度能从 −9.00 一直接到 PCB pad −4.10（约需 4.9mm 高的连接器或加焊脚/弹片）。同时把 `pads` 的 Z 改到配对 POGO 焊脚落点，而不是裸 PCB 背面。副作用：需选定真实配对 POGO 型号、核对其总高与凹腔深 `POGO_MATE_BODY_H`。
- **方案 2（针直触 pad）**：删除"配对 POGO"概念，把受电 pad 直接做在**后盖对接面外侧**（世界 Z≈−9.0 一带，针尖 −9.60 能压到），再用后盖内部走线/弹片把 pad 连到 PCB。副作用：后盖要走电、pad 不在 PCB 上了，PCBA 定义要改。
- **方案 3（加长针 + 抬高 pad）**：增大针凸出行程或缩短叠层，使针尖物理到达 PCB pad。副作用：5.5mm 行程的 POGO 针不现实；缩叠层会动整机厚度。

> 这是**架构级**问题，建议先与用户确认走"配对 POGO"还是"针直触"，再定参数。报告不擅自改。

### 🟠 B — 24P FPC 排座底缘撞后盖 plug 下沿 rim（装配阻断）

**证据：** `back_cover ∩ fpc = 10.50 mm³`，bbox Z[−5.6,−4.1]（= 排座本体 z 层）。

```
fpc 排座(局部) Y=[-47.50,-43.50]   ← 底缘到 PCB 边 (PCB_H/2=47.50)
元件包络半高 = 45.50
plug 中央镂空半高 = 46.50          ← 镂空只让到 -46.50
=> 排座底缘 -47.50 超出镂空 -46.50 共 1.0mm, 该 1mm 撞 plug 下沿 rim
```

**根因：** FPC 排座放在 `(0, −PCB_H/2+2.0)`（`enclosure_426.py:1095`），底缘到达板边 −47.50；而 plug 中央镂空 `BC_PLUG_CAV_H=93.0`（半高 46.5，`:350`）只让位到 −46.50。排座外伸的 1mm 落在 plug 下沿 rim（`BC_PLUG_RIM_H≈0.5`，`:353`）料上 → 板压不到位。

**可制造性结论：** 真干涉，会阻止后盖合拢/板落位。但注意排座尺寸 `FPC_CONN_W=14 / 深 4`（`:257`,`:1096`）是**估值**（§8 FPC 待测），实物可能更小。

**修复建议：**
- 短期：把 24P 排座往板内移（增大 `:1095` 的 +2.0 偏置到 ≥ 3.0），让其落在元件包络/镂空内；或
- 把 plug 下沿镂空局部开缺口让过排座（在 `make_back_cover` plug 镂空底边加一段 FPC 让位槽——本就是 `FPC_SLOT_ENABLE=False` 待 FPC 实测后要做的事，`:318`）。
- 根治：等 FPC/排座实测尺寸（§8），一并收敛排座位置与折回让位槽。副作用小。

### 🟡 C — POGO N/S 磁凸出顶住支架对接面薄壁（低，连接器估值）

**证据：** `bracket ∩ pogo_mag = 2.76 mm³`，bbox Z[−11.10,−10.90]（厚 0.2）。

**根因：** POGO 自带 N/S 磁建模为凸出本体顶面 `POGO_MAG_PROUD=0.2`（`enclosure_426.py:1129`）。支架对接面薄壁（`POGO_FACE_WALL=0.8`）只为 **4 根针**开了 Φ1.6 过孔（`:798-807`），没给 N/S 磁开让位 → 磁凸出的 0.2mm 顶在薄壁实料上。

**可制造性结论：** 真接触但量极小，且 `POGO_MAG_PROUD=0.2` 是图纸未注的估值。多半实物 N/S 磁与本体齐平、不凸出。

**修复建议：** 实测确认 POGO N/S 磁是否真凸出；若凸出，在薄壁对应 2 处开 Φ(磁+余量) 让位窗（与 4 针孔同层）。优先级低。

## 5. 修复优先级

1. **A（🔴 必须先定架构）** — POGO 受电链 5.5mm 空隙。**功能成败项**，需用户在"配对 POGO 桥接 / 针直触后盖外 pad"之间拍板，再改参数与 PCBA 定义。
2. **B（🟠 次之）** — 24P 排座撞 plug 下沿。可立即缓解（排座内移）或随 FPC 实测一并收敛折回让位槽。
3. **C（🟡 可缓）** — POGO N/S 磁凸出顶薄壁，待 POGO 实测确认是否真凸出再开让位窗。
4. **D/E/F（⚪ 清噪声，可选）** — 自检/可视化里去掉双重计入与飞线柱桩，让干涉矩阵更干净，避免未来误判。
